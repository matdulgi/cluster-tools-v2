# Usage
``` bash
cltls [parr] &lt;cmd> &lt;target> [args]
```

### e.g. Commands
- cltls cmd all ls -al  
- cltls cp all ~/goodday.txt

- cltls sync all 

### e.g. Parallel Commands

- cltls par cmd all ls -al
- cltls par cp all ~/goodday.txt
- cltls par sync all 

### e.g. Tools
- cltls ssh key exchange
- cltls hosts 

### Help
- cltls help
- cltls cmd help
- cltls sync help
